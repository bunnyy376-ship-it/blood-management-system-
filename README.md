# blood-management-system-
The Blood Management System is a web-based application designed to simplify and organize the process of blood donation and blood request management. The system connects donors, recipients, and blood banks through a centralized digital platform, reducing delays during emergency situations and improving blood availability tracking.
This project allows users to register as blood donors, search for blood groups, request blood during emergencies, and manage donor information efficiently. It provides an easy-to-use interface for maintaining records such as donor details, blood group availability, contact information, and request history.
The main objective of the system is to reduce manual paperwork and improve the speed, accuracy, and reliability of blood management operations. By digitizing the process, the application helps hospitals and patients quickly find suitable donors and available blood units.
